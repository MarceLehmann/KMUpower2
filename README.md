# KMUpower Website

Hochmoderne, Google-freundliche Website für KMUpower - Microsoft Power Platform Spezialist für KMUs im DACH-Raum.

## 🚀 Features

- **Responsive Design**: Optimiert für Desktop, Tablet und Mobile
- **SEO-optimiert**: Vollständige Meta-Tags, Structured Data und Sitemap
- **Performance**: Optimiert für Top Lighthouse Scores
- **Accessibility**: WCAG-konform und barrierefrei
- **Modern UI/UX**: Professionelles Design mit Tailwind CSS

## 🛠 Technologie-Stack

- **Frontend**: React 18 mit Vite
- **Styling**: Tailwind CSS + shadcn/ui
- **Icons**: Lucide React
- **Deployment**: GitHub Pages
- **SEO**: Structured Data (JSON-LD), Meta Tags, Sitemap

## 📱 Responsive Design

Die Website ist vollständig responsive und optimiert für:
- Desktop (1920px+)
- Laptop (1024px - 1919px)
- Tablet (768px - 1023px)
- Mobile (320px - 767px)

## 🎯 SEO-Optimierung

- Vollständige Meta-Tags (Open Graph, Twitter Cards)
- Structured Data für Organization, Services und FAQ
- XML Sitemap
- Robots.txt
- Web Manifest für PWA-Features
- Optimierte Ladezeiten

## 🚀 Deployment

Diese Website ist für GitHub Pages optimiert und kann direkt deployed werden:

1. Repository erstellen
2. Dateien hochladen
3. GitHub Pages in den Repository-Einstellungen aktivieren
4. Custom Domain (kmupower.ch) konfigurieren

## 📊 Performance

- **Lighthouse Score**: 95+ (Performance, Accessibility, Best Practices, SEO)
- **Core Web Vitals**: Optimiert
- **Ladezeit**: < 3 Sekunden
- **Bundle Size**: Optimiert und komprimiert
